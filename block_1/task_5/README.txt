Пример нахождения индекса Хирша на массиве чисел.
На вход подаётся текстовый документ input.txt со строкой из чисел, разделённых пробелом.
Необходимо найти индекс Хирша и вывести результат.
Для проверки есть программа test.py в которую можно вести результат программы main.py и она выведет количество чисел из input.txt больших или равных введённому.

Инструкция:
1) Вручную создаётся файл input.txt в соответствии с условием или запускается программа generator.py.
2) Запускается файл main.py.
3) Результат будет выведен на 5 секунд в терминале.
